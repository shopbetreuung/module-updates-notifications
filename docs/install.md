# Installation
1. Loggen Sie sich in den Admin-Bereich ein
2. Öffnen Sie die **Menüoption Module > Systemmodule**.
3. Wählen Sie das **Benachrichtigung bei Neuigkeiten** und klicken Sie auf Installieren.
